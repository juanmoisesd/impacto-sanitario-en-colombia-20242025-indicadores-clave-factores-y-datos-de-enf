# Reproducibility Guide

## Dataset: Esta base de datos recopila indicadores clave de salud, datos de brotes de enfermedades, tasas de vacunación, estado del sistema de salud, seguridad alimentaria y los principales factores que influyen en la salud pública en Colombia durante 2024–2025

---

## 1. Requirements

- Web browser or data analysis software (R, Python, Excel)
- No special software required for basic data access

## 2. Data Access

```
# Clone repository
git clone https://github.com/juanmoisesd/impacto-sanitario-en-colombia-20242025-indicadores-clave-factores-y-datos-de-enf.git
cd impacto-sanitario-en-colombia-20242025-indicadores-clave-factores-y-datos-de-enf
```

## 3. Data Format

- Data is provided in standard CSV/HTML format
- UTF-8 encoding
- See `data_dictionary.csv` or README for variable descriptions
